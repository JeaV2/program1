Wanneer gebruik je een For loop en wanneer gebruik je een While loop?

For loop

Je gebruikt een for loop wanneer het bekend is hoe vaak de loop moet worden uitgevoerd.
Bijvoorbeeld wanneer je een een stuk code 10 keer wilt uitvoeren, maar het stuk code niet 10 keer wil typen.

While loop

Je gebruikt een while loop wanneer het niet bekend is hoe vaak de loop moet worden uitgevoerd.
Bijvoorbeeld wanneer je een stuk code wilt uitvoeren totdat een bepaalde waarde overeenkomt.
